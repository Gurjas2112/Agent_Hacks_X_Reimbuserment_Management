Expense Management — Reimbursement System

Configurable approval workflows • OCR receipt scanning • Multi-currency conversion

🚀 Key Features

🔄 Configurable Approval Workflows

Four powerful approval modes:

  • Sequential — Step-by-step approval chain
  • Percentage — Threshold-based (e.g., 60% of approvers must approve)
  • Specific Approver — Auto-approve when a designated person (CFO) approves
  • Hybrid — Percentage OR specific approver — whichever triggers first

Optional "Manager First" step auto-inserts the employee's manager as Step 0.

🔍 OCR Receipt Scanning

Upload a receipt image and let AI extract structured data:

  • Vendor name, address, GST number
  • Transaction date and invoice number
  • Amounts (subtotal, tax, total) with currency detection
  • Line items auto-populated
  • Category auto-guessed from receipt content

Powered by DocStrange cloud OCR with regex fallback.

💱 Multi-Currency Support

Submit expenses in any currency:

  • Automatic conversion to company currency using ExchangeRate API v6
  • Real-time exchange rates with session caching
  • Country-to-currency auto-mapping from 250+ countries
  • Exchange rate recorded for audit trail

📋 Expense Policies

Enforce corporate spending rules:

  • Per-category maximum spending limits
  • Receipt requirements with configurable thresholds
  • Policy violation flags visible to approvers
  • Flexible per-company policy configuration

📊 Analytics & Reports

  • Pivot table analysis by category, employee, time period
  • Graph/chart views for expense distribution
  • Monthly spending trend analysis
  • Full audit log with approval history

👥 Role-Based Access

  • Employee — Submit expenses, view own history
  • Manager — Approve/reject team expenses, view reports
  • Admin — Full access: configure workflows, policies, users

Cascading security groups with implied permissions.

📧 Email Notifications

Automated HTML email notifications for:

  • New expense submitted — notification to approver
  • Expense approved — confirmation to employee
  • Expense rejected — notification to employee with guidance
